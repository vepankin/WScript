 -----------------------------------------------------
|   Описание шагов скрипта "DataExchange.js"      |
 -----------------------------------------------------

1.   Перезапуск сервиса сервера 1Сv81
2.   Создание копии информационной базы средствами 1С на локальном диске (.dt файл)
3.   Сохранение конфигурации на локальном диске (.cf файл)
4.   Перезапуск сервиса сервера 1Сv81
5.   Обновление конфигурации wtc_work_ku_81
6.   Реиндексация базы wtc_work_ku_81
7.   Копирование и удаление созданной ранее копии информационной базы с локального диска на сетевой (Q:)
8.   Копирование и удаление созданной ранее копии конфигурации с локального диска на сетевой (Q:)
9.   Выгрузка данных из "1С:Управление договорами" для КУ
10. Выгрузка данных из "1С:Совинтел" для КУ (обмен между УД и SI выполняется на сервере 1cv8wtc)
11. Загрузка данных в "1С:Корпоративный учет" из созданных на предыдущем шаге файлов выгрузки баз УД и Совинтел, выгрузка из КУ данных для баз УД и Совинтел
12. Загрузка данных в "1С:Управление договорами" из файлов, полученных на предыдущем шаге
13. Загрузка данных в "1С:Совинтел" из файлов, полученных на предыдущем шаге
14. Сопоставление справочников в УД: "Должности", "Подразделения", "Физические лица", синхронизация с данными, загруженными из КУ
15. Запуск выполнения внешних обработок в КУ из справочника "Обработки"
